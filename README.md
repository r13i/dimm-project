# dimm-project
Building a software for astronomic DIMM (Differential Image Motion Monitor)



### Requirements
- Python 3.7
- PyQt5 version 5.12 `pip install PyQt5==5.12`
- OpenCV version 3.4.5.20 `pip install opencv-python==3.4.5.20`
